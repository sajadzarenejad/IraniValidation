#An Iranian validation library for validating mobile number and national Id and some other values.

some example:

validate mobile number ==> "09190618636".IsValidMobile()    returns true

validate only irancell mobile numbers ==> "09190618636".IsValidMobile(OpratorType.IranCell)   returns false
